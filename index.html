Getting started:

Create a list:

var jaylist = new List();

Add some keys & values:

jaylist.add('this', 42); jaylist.add('that', 'red'); jaylist.add('other', 2.7); jaylist.add('thing', ['arrays', 'as', 'values']);

Get a value (42) via its key:

var val = jaylist.get('this');

Remove a key-value pair:

jaylist.remove('this');

About jaylist

By extending Javascript's built in hash storage, jaylist provides functions to manipulate an associative array. It accepts keys of any hashable type and associates them with any value. jaylist and its functions were inspired by Python's dictionary type. It is designed to be small (<1kb) and easy to use.

More Functions:

Get the keys in the list:

var keys = jaylist.keys(); keys === ['that', 'other', 'thing']

Get the values in the list:

var vals = jaylist.values(); vals === ['red', 2.7, ['arrays', 'as', 'values']]

Get the length of the list:

var len = jaylist.len(); len === 3

Check if the list contains the key:

var bool = jaylist.hasKey('this'); bool === false

Pop the key from the list:

var value = jaylist.pop('that'); value === 'red'

Pop an item from the list:

var pair = jaylist.popItem('thing'); pair === ['thing', ['arrays', 'as', 'values']]
