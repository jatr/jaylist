jaylist

a simple wrapper for javascript's hash storage.

create a list:

var jaylist = new List();

add to the list:

jaylist.add("this", 342);

get a value from the list:

jaylist.get("this"); // returns 342

overwrite a value in the list:

jaylist.add("this", 0);
jaylist.get("this"); // returns 0

remove a value from the list:

jaylist.remove("this");

get the keys from the list:

var keys = jaylist.keys();

get the values from the list:

var values = jaylist.values();
